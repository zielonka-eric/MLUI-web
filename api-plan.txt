GET     /api/help
    return list of algorithms, their types and parameters

GET     /api/help/adv
    return list of algorithms, their types and parameters,
        including advanced options

POST    /api/model
    args:
        data=<upload file or data_id of a file already uploaded>
        alg=<name of algorithm>
        <algorithm parameters - see the algorithm list for specific parameters>
        train=<column name to train on>
            Repeat this parameter for each column
        target=<column name to predict>
    return <model_id> of model to be created or error

POST    /api/model
    args:
        data=<upload file or data_id of a file already uploaded>
        alg-type=<class of algorithms: clustering|classification|regression >
        train=<column name to train on>
            Repeat this parameter for each column
        target=<column name to predict>
    return <model_id> of model to be created or error,
    uses the recommended model

GET     /api/model/<model_id>
    return creation status of model defined by <model_id>

GET     /api/model/<model_id>/download
    returns download of pickled model defined by <model_id>

POST    /api/model/<model_id>/test
    args:
        data=<upload file or data_id of a file already uploaded>
    tests the model defined by <model_id>
    returns the new result_id

GET     /api/model/<model_id>/results
    returns all of model's testing results

POST    /api/model/<model_id>/remove
    removes model defined by <model_id> from the database
    returns a confirmation that the model was removed

GET     /api/results/<result_id>
    returns one set of testing results defined by <result_id>

POST    /api/data
    args:
        data=<file to be uploaded>
    uploads data to the database to be used later
    returns new data_id

GET     /api/data/<data_id>
    return data defined by <data_id>

POST    /api/data/<data_id>/remove
    remove data defined by <data_id> from the database
    returns a confirmation that the data was removed
